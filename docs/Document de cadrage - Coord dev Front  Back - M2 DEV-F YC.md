## **1 Projet Jeu Vidéo** 

- **Module :** Développement d’application front/back 

- **Rendu :** Dépôt Github - **Dépôt public** 

- **Format :** Projet individuel 

- **À rendre pour le :** `10/07/2026` 

## **1.1 Objectif** 

Vous devez produire un **jeu vidéo fullstack Web** qui met en oeuvre une communication backend/frontend. 

- Tous les détails de votre projet (objectif, stack technique, implémentation de l’IA, etc.) ont été synthétisés et validés dans votre document de cadrage, produit lors de l’activité 4 

- Pas de déploiement exigé, une execution en local est suffisante 

- Tests et CI ne rentrent pas dans le périmètre du projet 

- Ce projet peut être utilisé en projet fil rouge Ynov Connect (validé par la pédagogie) 

Certains projets ne mettent pas en oeuvre un jeu vidéo, cela doit avoir été validé en amont par les formateurs et indiqué dans votre document de cadrage.
